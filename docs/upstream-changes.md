# 與上游專案的差異

MusicPlayer 基於 [twtrubiks/lyra-music](https://github.com/twtrubiks/lyra-music) 開發，保留其 Tauri、Svelte、Rust 與 SQLite 架構，以及 MIT 授權。

以下項目為本專案相較上游的主要方向，尚未完成的功能仍屬規劃。

## 核心功能比較

> ✅ 支援　❌ 不支援　⚠️ 部分支援。MusicPlayer 欄位包含規劃功能。

| 核心功能 | 上游 Lyra Music | MusicPlayer Lite | MusicPlayer 完整版 |
|----------|:---------------:|:----------------:|:------------------:|
| 本機音樂播放 | ✅ | ✅ | ✅ |
| 離線媒體庫與播放清單 | ✅ | ✅ | ✅ |
| Album 瀏覽與管理 | ✅ | ❌ | ❌ |
| 多 Tag 分類 | ❌ | ✅ | ✅ |
| Artist / Tag 播放操作 | ⚠️ | ✅ | ✅ |
| Mini Player 與 System Tray | ✅ | ✅ | ✅ |
| Windows 工作列播放器 | ❌ | ✅ | ✅ |
| MP4 視訊播放 | ❌ | ❌ | ✅ |
| 動態音樂波形 | ❌ | ❌ | ✅ |
| YouTube 下載 | ❌ | ❌ | ✅ |
| 不需帳號或遙測 | ✅ | ✅ | ✅ |
| 不連網即可使用核心功能 | ✅ | ✅ | ✅ |

## 產品方向

- 以 Windows 作為主要設計與使用平台，同時盡量維持 macOS / Linux 支援。
- 分為 Lite 與完整版，使用者可依功能與資源需求選擇。
- 不依賴帳號或遙測，本機媒體庫與播放功能皆可離線使用。

## 兩個版本的共用差異

- 移除 Album 瀏覽與管理功能，改為可讓同一媒體擁有多個分類的 Tag 系統。
- 擴充 Artist 與 Tag 的播放、隨機播放、加入佇列及播放清單操作。
- 新增 Windows 工作列播放器，提供精簡的播放資訊與控制。
- 保留播放清單、Mini Player、System Tray 與輕量背景播放能力。

## Lite 版

- 專注於純本機音樂播放、媒體庫、Tag 與播放清單。
- 不支援下載、MP4 視訊播放及動態音樂波形。
- 以較小的安裝體積、快速啟動與低資源佔用為目標。

## 完整版

- 包含 Lite 版所有功能。
- 支援 MP4 視訊播放，並可由使用者選擇是否顯示畫面。
- 支援可開關的動態音樂波形。
- 支援 YouTube 單一影片與播放清單下載，並可匯入媒體庫。
